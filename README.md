## 常见问题

### 什么是代理服务器？
代理服务器是在您的计算机与互联网之间起到中介作用的一台服务器。当您连接到互联网时，实际上是先连接到代理服务器，然后代理服务器再连接到您想访问的网站或服务。代理服务器可用于多种目的，例如提升安全性或性能，并确保采集准确的客户端数据点。

### 为什么要使用代理服务器？
当您在防火墙后面，需要访问某些网站时，就可能需要使用代理服务器。它充当您的设备与互联网之间的中介，帮助您访问可能被限制或显示不准确信息的网站。
